# Tic Tac Toe

A browser-based Tic Tac Toe game created as part of The Odin Project curriculum.

## Features
- Player vs Player gameplay
- Custom player names
- Interactive game board
- Win/Draw detection
- Game restart functionality

## Technologies Used
- HTML
- CSS
- JavaScript
